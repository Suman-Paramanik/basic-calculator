# Basic Calculator

## 📌 Overview

This is a simple and responsive Basic Calculator built using HTML, CSS, and JavaScript. The calculator performs common arithmetic operations with a clean and user-friendly interface. It is designed to help users perform quick mathematical calculations directly in the browser.

##  Features

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Percentage (%)
* Decimal Number Support
* Clear (AC) Button
* Delete (DEL) Button
* Responsive Design
* Keyboard-Friendly Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How It Works

1. Open the calculator in your web browser.
2. Click the number buttons to enter values.
3. Select an arithmetic operator (+, -, ×, ÷).
4. Enter the second number.
5. Press the "=" button to view the result.
6. Use **AC** to clear all inputs or **DEL** to remove the last entered character.
